R2C: Mapping Room to Chessboard to Unlock LLM As Low-Level Action Planner

Ziyi Bai*, Hanxuan Li*, Bin Fu*, Chuyan Xiong, Ruiping Wang, Xilin Chen
Institute of Computing Technology, Chinese Academy of Sciences
*Indicates Equal Contribution
Paper Supplementary Code

We unlock LLM as low-level action planner by mapping the room into chessboard, which we call Room to Chessboard (R2C). It provides a communication platform between LLM and Robot. Robot can convey room state to LLM, and LLM can perform low-level action plannning by playing chess.

Abstract

LLMs have been proved to be a general high-level task planner for diverse embodied tasks. However, there exists a natural gap between LLM and the low-level executor. LLM cannot sufficiently perceives the whole complex environment state, often leads to unavailable and unattainable plans that robot cannot follow. In this work, we achieve high-level and low-level planning in an integrated manner by providing a communication platform. Our insight is to formulate the task as playing chess with LLMs. We map the room into a semantic chessboard, which we call Room to Chessboard (R2C). Each grid represents the position and size of objects inside the room. We find that chessboard is concise enough for LLMs to conduct semantic searches with global view of the room. Also, the chessboard is informative enough to convey detailed environment state for LLMs to predict executable low-level actions. To enhance LLM's low-level decision-making capacity, we also design a Chain-of-Thought fine-tuning paradigm to guide LLMs make more interpretable planning. We implement the R2C framework with both fine-tuned open-sourced LLMs and close-sourced LLMs like GPT-4. Evaluations on challenging ALFRED benchmark indicate that using LLMs as low-level action planners is feasible by efficiently communication between LLM and the robot. We also demonstrate that R2C is a versatile framework for diverse free-form robotic planning tasks.

The comparison between LLM as a high-level planner and as a low-level planner. Upper: most existing methods rely solely on LLM as a high-level planner. the robot cannot effectively convey spatial information to LLM, resulting in LLM only being able to perform basic task decomposition; Bellow: the R2C framework transforms the real environment into a chessboard that both LLM and the robot can understand, turning embodied tasks into a chess game. LLM can then engage in low-level planning based on the grid information of the chessboard.

Overview of R2C: After high-level planning, environmental information is transformed into a grid-based chessboard by the Environment Filter. Subsequently, this textual chessboard, along with the task rules, forms the prompt input for either a zero-shot or CoT fine-tuned LLM. Finally, the LLM predicts the next position, which is then executed by the robot to obtain new visual information, thereby completing the loop.

Case study for R2C. The red lines on the chessboard are trajectories predicted by LLM, where each key point is marked with circles. We visualize the HLP and LLP process at steps 8, 15, and 34 in detail.
